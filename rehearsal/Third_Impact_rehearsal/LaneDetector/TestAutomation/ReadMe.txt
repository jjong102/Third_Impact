This directory contains the following files:

1. LaneMarkerDetectorTestAssessments.mldatx [Simulink test file]
2. LaneMarkerDetectorTestAssessments.slmx   [Traceability data file]
3. LaneMarkerDetectorTestRequirements.slreqx  [Requirements data file]

These files are related to automate testing of lane marker detector test bench model. If you have the licenses to Simulink Requirements and Simulink Test, then these files can be used to run the test scenarios in automated way.

For more details, open the example using:
openExample('autonomous_control/AutomateTestingForLaneMarkerDetectorExample');
